# E-BILLING ISP SYSTEM

**Sistem E-Billing Terintegrasi untuk Provider ISP/Telecom Indonesia**

## Project Overview

Aplikasi manajemen billing lengkap dengan integrasi:
- Payment Gateway (Xendit, Tripay, Moota)
- WhatsApp Gateway (Otomasi notifikasi)
- Mikrotik API (Blokir/unlock pelanggan)
- Ticketing System
- Inventory Management
- Payroll & HR
- Financial Reporting

## Tech Stack

- **Backend**: Laravel 11 LTS + PostgreSQL
- **Frontend**: React 18
- **Mobile**: React Native + Expo
- **DevOps**: Docker + Docker Compose
